# Customer Churn Analysis Dashboard
This project analyzes customer churn using Excel (Power Query, Pivot Tables, and interactive charts).
The aim is to identify the main factors that drive customers to leave and help the business take data-driven actions to improve retention.

## Features
* Data cleaning and transformation using **Power Query**
* Interactive **Churn Dashboard**
* KPIs:
  * Total Revenue
  * Lost Revenue
  * Monthly Charges
  * Overall Churn Rate

## Key Insights
* Customers without Tech Support have significantly higher churn.
* Higher monthly charges are associated with higher churn.
* Electronic Check is the most used payment method and the one with the highest churn.
* Senior Citizens show slightly different churn behavior but not strongly correlated.
* Contract type strongly affects churn — month-to-month contracts have the highest rate.

## Tools Used
* Microsoft Excel
* Power Query
* Pivot Tables & Charts

## Purpose & Backround
This project was created as a practical application of the concepts from the book “Thinking with Data”.
The goal was to translate the analytical thinking framework from the book into a real-world business problem — understanding the drivers of customer churn — and building a dashboard that supports data-driven decisions.
